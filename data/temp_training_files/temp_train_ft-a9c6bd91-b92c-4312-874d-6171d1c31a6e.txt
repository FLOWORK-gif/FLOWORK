apa itu flowork
Flowork adalah Platform Orkestrasi Alur Kerja Visual yang memungkinkan pengguna merancang, membangun, dan menjalankan proses otomatis secara visual dengan antarmuka "seret dan lepas" (drag-and-drop).
bagaimana cara kerja flowork
Flowork bekerja dengan menggunakan kanvas visual di mana pengguna menempatkan 'node' (modul atau plugin) dan menghubungkannya untuk menciptakan alur kerja. Setiap node memiliki satu tugas spesifik. Alur kerja ini kemudian dapat dieksekusi baik melalui antarmuka grafis, dari command-line (headless), atau dipicu oleh API eksternal.
apa itu node di flowork
Node adalah blok bangunan dasar dari sebuah alur kerja di Flowork. Setiap node mewakili satu unit tugas, seperti membaca file, mengirim email, atau menjalankan logika if/else. Node berasal dari Modul atau Plugin yang terinstal.
apa perbedaan modul dan plugin
Secara fungsional, keduanya sama-sama merupakan node yang bisa digunakan di kanvas. Perbedaannya hanya pada kategorisasi di dalam toolbox untuk memudahkan pengguna. Modul biasanya untuk 'Logic' (logika internal alur kerja), sedangkan Plugin untuk 'Action' (berinteraksi dengan sistem luar).
bagaimana cara menyimpan alur kerja
Kamu bisa menyimpan alur kerja dengan mengklik tombol "Save Workflow as Preset" di atas area kanvas. Kamu akan diminta memasukkan nama untuk preset tersebut, yang kemudian akan disimpan sebagai file .json.
apa itu webhook di flowork
Webhook adalah sebuah URL unik yang disediakan oleh Flowork untuk setiap preset. Ketika URL ini diakses dari aplikasi atau layanan lain dengan metode POST, alur kerja yang terkait akan otomatis berjalan. Ini memungkinkan integrasi dengan sistem eksternal.
bagaimana cara menjalankan alur kerja tanpa membuka aplikasi
Kamu bisa menjalankannya dalam mode headless melalui command-line. Gunakan perintah seperti 'python main.py --run-preset NAMA_PRESET_ANDA'. Ini akan mengeksekusi alur kerja tanpa menampilkan antarmuka pengguna.
apa fungsi modul Advanced Web Scraper
Modul Advanced Web Scraper digunakan untuk mengekstrak data dari situs web modern yang menggunakan JavaScript. Modul ini mengontrol browser secara otomatis (menggunakan Selenium) untuk memuat halaman sepenuhnya sebelum mengambil data berdasarkan aturan CSS selector yang diberikan.
apa itu AI Architect
AI Architect adalah fitur premium yang memungkinkan kamu membuat kerangka alur kerja secara otomatis hanya dengan menuliskan perintah dalam bahasa natural. AI akan menganalisis permintaanmu dan membuat susunan node yang relevan di kanvas.
bagaimana cara menambahkan variabel global
Variabel global atau 'secret' dapat ditambahkan melalui menu Settings > Variable & Secret Management. Variabel yang dibuat di sana dapat dipanggil di dalam node mana pun menggunakan format placeholder, contohnya {{vars.NAMA_VARIABEL_SAYA}}.
apa kegunaan widget di dashboard
Widget di dashboard menyediakan alat bantu dan tampilan informasi dalam satu layar. Contohnya adalah Toolbox untuk memilih modul, Log Viewer untuk melihat log eksekusi, dan Dataset Manager untuk mengelola data pelatihan AI. Layout widget dapat diatur dan disimpan sesuai keinginan.
apa itu Time-Travel Debugger
Time-Travel Debugger adalah fitur premium yang muncul setelah alur kerja selesai dieksekusi (jika diaktifkan). Fitur ini memungkinkan kamu untuk "mundur" ke setiap langkah koneksi dalam alur kerja dan melihat data (payload) yang lewat pada saat itu, sangat berguna untuk melacak kesalahan atau memahami alur data.
bagaimana cara membuat modul flowork sendiri
Kamu bisa menggunakan fitur "Module Generator" yang ada di dalam tab AI Tools. Fitur ini menyediakan antarmuka visual untuk merancang properti UI dan logika 'execute' dari sebuah modul, lalu mengemasnya menjadi file .zip yang siap diinstal.
apa itu GGUF
GGUF adalah format file yang digunakan untuk menyimpan dan menjalankan model bahasa besar (seperti Llama) secara efisien di komputer lokal. Flowork mendukung penggunaan model GGUF sebagai 'otak' untuk AI Agent atau untuk tugas-tugas AI lainnya.
apa fungsi modul Keyword Router
Modul Keyword Router berfungsi seperti percabangan if/else. Ia akan memeriksa teks input dan mengarahkan alur kerja ke port output yang berbeda berdasarkan kata kunci yang ditemukan. Misalnya, jika teks mengandung kata "error", alur akan diarahkan ke port 'penanganan_error'.

Modul Advanced Web Scraper digunakan untuk mengekstrak data dari situs web modern yang menggunakan JavaScript. Modul ini mengontrol browser secara otomatis (menggunakan Selenium) untuk memuat halaman sepenuhnya sebelum mengambil data berdasarkan aturan CSS selector yang diberikan.
apa itu AI Architect
AI Architect adalah fitur premium yang memungkinkan kamu membuat kerangka alur kerja secara otomatis hanya dengan menuliskan perintah dalam bahasa natural. AI akan menganalisis permintaanmu dan membuat susunan node yang relevan di kanvas.
bagaimana cara menambahkan variabel global
Variabel global atau 'secret' dapat ditambahkan melalui menu Settings > Variable & Secret Management. Variabel yang dibuat di sana dapat dipanggil di dalam node mana pun menggunakan format placeholder, contohnya {{vars.NAMA_VARIABEL_SAYA}}.
apa kegunaan widget di dashboard
Widget di dashboard menyediakan alat bantu dan tampilan informasi dalam satu layar. Contohnya adalah Toolbox untuk memilih modul, Log Viewer untuk melihat log eksekusi, dan Dataset Manager untuk mengelola data pelatihan AI. Layout widget dapat diatur dan disimpan sesuai keinginan.
apa itu Time-Travel Debugger
Time-Travel Debugger adalah fitur premium yang muncul setelah alur kerja selesai dieksekusi (jika diaktifkan). Fitur ini memungkinkan kamu untuk "mundur" ke setiap langkah koneksi dalam alur kerja dan melihat data (payload) yang lewat pada saat itu, sangat berguna untuk melacak kesalahan atau memahami alur data.
bagaimana cara membuat modul flowork sendiri
Kamu bisa menggunakan fitur "Module Generator" yang ada di dalam tab AI Tools. Fitur ini menyediakan antarmuka visual untuk merancang properti UI dan logika 'execute' dari sebuah modul, lalu mengemasnya menjadi file .zip yang siap diinstal.
apa itu GGUF
GGUF adalah format file yang digunakan untuk menyimpan dan menjalankan model bahasa besar (seperti Llama) secara efisien di komputer lokal. Flowork mendukung penggunaan model GGUF sebagai 'otak' untuk AI Agent atau untuk tugas-tugas AI lainnya.
apa fungsi modul Keyword Router
Modul Keyword Router berfungsi seperti percabangan if/else. Ia akan memeriksa teks input dan mengarahkan alur kerja ke port output yang berbeda berdasarkan kata kunci yang ditemukan. Misalnya, jika teks mengandung kata "error", alur akan diarahkan ke port 'penanganan_error'.
apa itu flowork
Flowork adalah Platform Orkestrasi Alur Kerja Visual yang memungkinkan pengguna merancang, membangun, dan menjalankan proses otomatis secara visual dengan antarmuka "seret dan lepas" (drag-and-drop).
bagaimana cara kerja flowork
Flowork bekerja dengan menggunakan kanvas visual di mana pengguna menempatkan 'node' (modul atau plugin) dan menghubungkannya untuk menciptakan alur kerja. Setiap node memiliki satu tugas spesifik. Alur kerja ini kemudian dapat dieksekusi baik melalui antarmuka grafis, dari command-line (headless), atau dipicu oleh API eksternal.
apa itu node di flowork
Node adalah blok bangunan dasar dari sebuah alur kerja di Flowork. Setiap node mewakili satu unit tugas, seperti membaca file, mengirim email, atau menjalankan logika if/else. Node berasal dari Modul atau Plugin yang terinstal.
apa perbedaan modul dan plugin
Secara fungsional, keduanya sama-sama merupakan node yang bisa digunakan di kanvas. Perbedaannya hanya pada kategorisasi di dalam toolbox untuk memudahkan pengguna. Modul biasanya untuk 'Logic' (logika internal alur kerja), sedangkan Plugin untuk 'Action' (berinteraksi dengan sistem luar).
bagaimana cara menyimpan alur kerja
Kamu bisa menyimpan alur kerja dengan mengklik tombol "Save Workflow as Preset" di atas area kanvas. Kamu akan diminta memasukkan nama untuk preset tersebut, yang kemudian akan disimpan sebagai file .json.
apa itu webhook di flowork
Webhook adalah sebuah URL unik yang disediakan oleh Flowork untuk setiap preset. Ketika URL ini diakses dari aplikasi atau layanan lain dengan metode POST, alur kerja yang terkait akan otomatis berjalan. Ini memungkinkan integrasi dengan sistem eksternal.

GGUF adalah format file yang digunakan untuk menyimpan dan menjalankan model bahasa besar (seperti Llama) secara efisien di komputer lokal. Flowork mendukung penggunaan model GGUF sebagai 'otak' untuk AI Agent atau untuk tugas-tugas AI lainnya.
apa fungsi modul Keyword Router
Modul Keyword Router berfungsi seperti percabangan if/else. Ia akan memeriksa teks input dan mengarahkan alur kerja ke port output yang berbeda berdasarkan kata kunci yang ditemukan. Misalnya, jika teks mengandung kata "error", alur akan diarahkan ke port 'penanganan_error'.
apa itu flowork
Flowork adalah Platform Orkestrasi Alur Kerja Visual yang memungkinkan pengguna merancang, membangun, dan menjalankan proses otomatis secara visual dengan antarmuka "seret dan lepas" (drag-and-drop).
bagaimana cara kerja flowork
Flowork bekerja dengan menggunakan kanvas visual di mana pengguna menempatkan 'node' (modul atau plugin) dan menghubungkannya untuk menciptakan alur kerja. Setiap node memiliki satu tugas spesifik. Alur kerja ini kemudian dapat dieksekusi baik melalui antarmuka grafis, dari command-line (headless), atau dipicu oleh API eksternal.
apa itu node di flowork
Node adalah blok bangunan dasar dari sebuah alur kerja di Flowork. Setiap node mewakili satu unit tugas, seperti membaca file, mengirim email, atau menjalankan logika if/else. Node berasal dari Modul atau Plugin yang terinstal.
apa perbedaan modul dan plugin
Secara fungsional, keduanya sama-sama merupakan node yang bisa digunakan di kanvas. Perbedaannya hanya pada kategorisasi di dalam toolbox untuk memudahkan pengguna. Modul biasanya untuk 'Logic' (logika internal alur kerja), sedangkan Plugin untuk 'Action' (berinteraksi dengan sistem luar).
bagaimana cara menyimpan alur kerja
Kamu bisa menyimpan alur kerja dengan mengklik tombol "Save Workflow as Preset" di atas area kanvas. Kamu akan diminta memasukkan nama untuk preset tersebut, yang kemudian akan disimpan sebagai file .json.
apa itu webhook di flowork
Webhook adalah sebuah URL unik yang disediakan oleh Flowork untuk setiap preset. Ketika URL ini diakses dari aplikasi atau layanan lain dengan metode POST, alur kerja yang terkait akan otomatis berjalan. Ini memungkinkan integrasi dengan sistem eksternal.
bagaimana cara menjalankan alur kerja tanpa membuka aplikasi
Kamu bisa menjalankannya dalam mode headless melalui command-line. Gunakan perintah seperti 'python main.py --run-preset NAMA_PRESET_ANDA'. Ini akan mengeksekusi alur kerja tanpa menampilkan antarmuka pengguna.
apa fungsi modul Advanced Web Scraper
Modul Advanced Web Scraper digunakan untuk mengekstrak data dari situs web modern yang menggunakan JavaScript. Modul ini mengontrol browser secara otomatis (menggunakan Selenium) untuk memuat halaman sepenuhnya sebelum mengambil data berdasarkan aturan CSS selector yang diberikan.
apa itu AI Architect
AI Architect adalah fitur premium yang memungkinkan kamu membuat kerangka alur kerja secara otomatis hanya dengan menuliskan perintah dalam bahasa natural. AI akan menganalisis permintaanmu dan membuat susunan node yang relevan di kanvas.
bagaimana cara menambahkan variabel global
Variabel global atau 'secret' dapat ditambahkan melalui menu Settings > Variable & Secret Management. Variabel yang dibuat di sana dapat dipanggil di dalam node mana pun menggunakan format placeholder, contohnya {{vars.NAMA_VARIABEL_SAYA}}.
apa kegunaan widget di dashboard
Widget di dashboard menyediakan alat bantu dan tampilan informasi dalam satu layar. Contohnya adalah Toolbox untuk memilih modul, Log Viewer untuk melihat log eksekusi, dan Dataset Manager untuk mengelola data pelatihan AI. Layout widget dapat diatur dan disimpan sesuai keinginan.
apa itu Time-Travel Debugger
Time-Travel Debugger adalah fitur premium yang muncul setelah alur kerja selesai dieksekusi (jika diaktifkan). Fitur ini memungkinkan kamu untuk "mundur" ke setiap langkah koneksi dalam alur kerja dan melihat data (payload) yang lewat pada saat itu, sangat berguna untuk melacak kesalahan atau memahami alur data.
